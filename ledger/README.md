- 金额，正数代表收入，负数代表支出。

### 账目类别

一条账目具有类别属性，即这条账目属于什么类别的花费。

#### 类别具有树形结构的层级关系

【人际关系】>【生日礼物】>【水果】>【苹果】

【食】>【水果】>【苹果】

这里的两个【苹果】分类的含义是不同的，算两个分类，即不同分支下是允许重名的。

#### 类别与账目的关系是一对多的关系

一条账目的类别是最低级的类别（叶子结点）
